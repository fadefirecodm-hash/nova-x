Version 9.0 — Offline AI
Mar 2026
NEW
WebGPU Local AI (WebLLM) — run Qwen2-0.5B, Llama 3.2, Phi-3.5, or Gemma 2 fully on your GPU. No API key. No internet after download. Note tools (summarise, quiz, keywords etc.) switch to local automatically
NEW
Model preference synced via Gist — GitHub Gist stores which model you chose so it follows you across devices (each device downloads weights once)
NEW
Warm Cache — one tap pre-runs common study prompts while online so they're ready offline immediately
NEW
Cache Panel in Settings — see how many responses are cached, how much space used, and push/clear with one tap
PERF
WebGPU shader caching — compiled shaders stored in Cache API so subsequent loads skip recompilation entirely
PERF
Auto-boot — if a model was loaded last session it silently reloads into GPU memory 4 seconds after app start
FIX
Gist snapshot version 3 — upgraded sync format includes AI cache; fully backward compatible with v2 snapshots
